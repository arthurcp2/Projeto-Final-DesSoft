# Insper-Olimpics
# Objetivo do jogo:
Haverá uma competição olímpica que você é um outro candidato vão disputar, haverão 3 provas: corrida de 100 metros, salto em distância e lançamento de dardo. O vencedor será quem ganhar 2 dos 3 jogos.
# Instruções do jogo:
Para clonar o jogo deve além de copiar o código, copiar e baixar os arquivos "Font", " Music" e "Spritesheets". Por fim é necessário apenas rodar o código no "olympics.py".

O jogador clicar na tecla para começar os jogos, o primeiro jogo, corrida de 100 metros, e uma prova em que para correr deve-se ficar pressionando uma tecla constantemente, quanto mais rápido a tecla for pressionada, mais rápido o atleta anda. O jogador 1 pressiona a tecla "a" e o jogador 2 pressiona "l".

No segundo jogo existem duas teclas funcionais, uma para correr e outra para saltar, o jogador 1 pressiona "a" para correr e "espaço" para saltar, já o jogador 2, "l" para correr e "enter" para saltar. Nessa modalidade se o atleta encostar na linha está desqualificado e existe um delay para o salto.

Por fim o lançamento de dardo possui dois mecanismos, o jogador aperta uma tecla para definir a velocidade que o atleta corre e outra para definir o ângulo formado entre o chão e o lançamento da vara, o jogador 1 pressiona "a" para definir a velocidade e "a" para definir o ângulo e o jogador 2 "l" para velocidade e "l" para o ângulo
